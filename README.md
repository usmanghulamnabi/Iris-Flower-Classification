# 🌸 Iris Flower Classification Project

A machine learning project that classifies iris flowers into three species based on their measurements.

**Author: Dr. Muhammad Usman**

## 🔍 Overview

This project implements a K-Nearest Neighbors classifier to identify iris flower species. It's a classic project in machine learning that covers:

- Data loading and exploration
- Data preprocessing (scaling, encoding)
- Model training with hyperparameter tuning
- Model evaluation
- Saving/loading trained models

## 📊 Dataset

The famous Iris dataset contains 150 samples with the following attributes:

| Attribute | Description |
|-----------|-------------|
| Sepal Length | Length of sepal in cm |
| Sepal Width | Width of sepal in cm |
| Petal Length | Length of petal in cm |
| Petal Width | Width of petal in cm |
| Species | Target variable (Setosa, Versicolor, Virginica) |

## ✨ Features

- ✅ **Automatic Model Training**: Trains on first run, saves for later use
- ✅ **Smart Model Loading**: Loads existing model to avoid retraining
- ✅ **Interactive Mode**: Real-time predictions for custom measurements
- ✅ **Optimal k Selection**: Automatically finds the best k value for KNN
- ✅ **Probability Scores**: Shows confidence level for each prediction
- ✅ **Clean Output**: User-friendly console interface
